# About This Project

This page covers the origin story, acknowledgments, references, and license of the passage-of-time-mcp project.

## The Story

This project was created by [Jeremie Lumbroso](https://github.com/jlumbroso) and Claude Opus 4.0 through human-LLM collaboration. It started from a simple question — "Can AI perceive the passage of time?" — and became a practical MCP server that equips language models with time calculation tools. With proper temporal tools, models can uncover surprising insights about conversation patterns, work rhythms, and the human experience of time.

[Read the full story on Medium →](https://medium.com/@jeremie.lumbroso/teaching-ai-the-significance-of-the-passage-of-time-yes-that-one-106ad7d20957)
